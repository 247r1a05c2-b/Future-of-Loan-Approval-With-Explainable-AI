# 📊 Future of Loan Approvals with Explainable AI

## 🚀 Project Overview

This project presents an intelligent and transparent loan approval system using **Machine Learning** and **Explainable AI (XAI)**.

It predicts whether a loan will be approved or rejected and also explains **why** the decision was made using SHAP.

---

## 🎯 Key Features

* 📂 Upload real-world loan datasets
* ⚙️ Data preprocessing (handling missing values, encoding, normalization)
* 🤖 Loan approval prediction using Machine Learning
* ❌ Rejection reason prediction
* 📈 Performance metrics:

  * Accuracy
  * Precision
  * Recall
  * F1-Score
* 📊 Confusion Matrix visualization
* 🧠 Explainable AI using SHAP
* 🖥️ User-friendly GUI built with Tkinter

---

## 🏗️ System Architecture

1. Dataset Upload
2. Data Preprocessing

## ⚙️ Installation & Setup

### 1️⃣ Install Python

Download and install Python (version 3.7 or above)

### 2️⃣ Install Required Libraries

```
pip install numpy pandas matplotlib seaborn scikit-learn shap
```

### 3️⃣ Run the Project

```
python main.py
```

---

## ▶️ How to Use

1. Click **Upload Loan Application Dataset**
2. Click **Preprocess Dataset**
3. Click **Split Dataset Train & Test**
4. Click:

   * Train AI on Loan Approval
   * Train AI on Loan Rejections
5. Click **Explainable AI**
6. Upload test data and click **Predict**

📊 Future of Loan Approvals with Explainable AI
🚀 Project Overview

This project presents an intelligent and transparent loan approval system using Machine Learning and Explainable AI (XAI). It predicts whether a loan will be approved or rejected and also explains why the decision was made.

The system uses a Random Forest Classifier for prediction and SHAP (SHapley Additive Explanations) to provide interpretability.
